dumper

Dumper node module

Installation

$ npm install --save-dev dumper

Usage

ES5
var dumper = require('dumper').dumper;

var foo = { bar: 'baz' };

dumper(foo);
ES6
import {dumper} from 'dumper';

let foo = { bar: 'baz' };

dumper(foo);
